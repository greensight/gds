# Spacing

Для задания размеров элементов, полей и отступов используются шкалы major и minor - по 8 и по 4 px соответственно. Для их использования прописаны хелперы `major` и `minor`.

Примеры:

```js
<SomeComponent
    css={{
        width: major(20),
        height: major(6),
        padding: `${minor(1)}px ${major(1)}px`,
        marginBottom: major(2),
    }}
/>
```

Общие правила:

1. Использовать major/minor для всех размеров, где имеет место сетка: padding - хорошо, borderRadius - пишем в px.
1. Применять minor только для размеров, которые невозможно представить через major (некратные 8): major(2) - хорошо, minor(4) - бессмысленно.
1. minor не определён для размеров выше 40px. Хелпер выведет предупреждение и округлит результат до 8.
1. В качестве множителя принимаются только целые значения. Хелпер выведет предупреждение и округлит множитель до целого.

Для использования шага, отличного от 8, нужно определить majorStep в конфиге и переопределить хелпер с передачей своего объекта темы:

```js
import major from '@greensight/gds';
import theme from './theme';

const customMajor = multiplier => major(multiplier, theme.app);
```
