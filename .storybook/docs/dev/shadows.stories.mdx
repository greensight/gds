# Shadows

✨ Theme: `shadows`

✨ Tokens: `shadows`

✨ Autokit: `Shadows`

В переменных теней хрянятся значения для передачи в CSS правило `box-shadow`.

⛔️ Если вы используете токены, то не должны сами добавлять переменные. Если появилась такая необходимость, то вероятнее всего, что это ошибка макета. Обсуждайте такие моменты с ведущим дизайнером проекта.

**Интерфейс:**

```js
interface IShadows {
    /** Переменная тени */
    [name: string]: string;
}
```

## Использование

Нужно задать тень для некоторого элемента. В макете у этого элемента указан стиль:

<img src="images/shadows/style.png" alt="Стиль тени в Figma" width="300" />

_Стиль тени в Figma_

Вы знаете имя стиля и можете обратиться к соответствующей переменной темы:

```js
import { useTheme } from '@greensight/gds';

const Example = () => {
    const { shadows } = useTheme();
    return <div css={{ boxShadow: shadows.inner }}>Example</div>;
};
```
