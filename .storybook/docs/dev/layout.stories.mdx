# Layout

## Компоненты

Для упрощения работы с сетками GDS предоставляет несколько компонентов:

-   `Layout` - компонент для построения сеток на гридах и флексах
-   `Section` - компонент для вставки раздела в основное тело страницы
-   `Container` - компонент для добавления обёртки содержимого в контейнер

Подробнее об этих компонентах смотри в их документации.

## Токены

Каждый проект идёт с базовой сеткой, определённой в Figma во фрейме Layout. Эта сетка может соблюдаться или не соблюдаться в дальнейшем, но её значения проставляются во вспомогательные компоненты по умолчанию. Эти параметры:

-   `breakpoints` - список брейкпоинтов. Самый важный параметр, т.к. влияет не только на сетки, и задаёт значения для переменных для всего проекта;
-   `cols` - число колонок. Ставится в проп `cols` компонента `Layout` по умолчанию;
-   `gap` - отступы между колонками. Ставится в проп `gap` компонента `Layout` по умолчанию;
-   `container` - ширина контейнера (если есть). Ширина прописывается в компонент `Container` и от её наличия зависит начальное значение пропса `container` у компонента `Section`;
-   `padding` - поля основного контента. Размеры прописываются в компонент `Container`.
