# Colors

## Палитры проекта

Палитры получаются синхронизацией дизайн-токенов из фрейма `Palettes` Figma - дизайнер использует эти стаки оттенков для формирования стилей цветов.

⛔ Палитры непосредственно на проекте не используются. Они нужны на этапе дизайна и для демонстрации.

Так палитры хранятся в объекте темы:

```json
"palettes": {
    "grey": ["#121314", "#242529", "#35383d", "#474a52", "#595d66", "#6b6f7b", "#7e828e", "#9296a0", "#a0a3ac", "#adb0b8", "#bbbdc4", "#c9cbd0", "#d6d8db", "#e4e5e7", "#f1f2f3"],
    "blue": ["#021540", "#042471", "#0835a1", "#1146cc", "#215bf0", "#6391ff", "#90b2ff", "#c2d6ff", "#d9e4ff", "#eff2fa"],
    "red": ["#470000", "#6f0101", "#960202", "#b80303", "#d40505", "#ea1414", "#f14646", "#f57979", "#f5aaaa", "#f2dada"],
}
```

## Цвета проекта

Цвета получаются синхронизацией дизайн-токенов из фрейма `Colors` Figma - это финальные переменные, на которые заводятся стили. Привязка идёт по исполняемым функциям, значения берутся из палитры.

☝🏿 Одно и то же значение может быть привязано к разным переменным - первична функция, а не цвет.

Цвета разбиваются на группы:

1. **text** - цвета текста
1. **bg** - цвета фона
1. **border** - цвета рамок и разделителей
1. **function** - функциональные цвета (успех, ошибка, предупреждение)

Так цвета хранятся в объекте темы:

```json
"colors": {
    "text": { link: "#215bf0", linkHover: "#1146cc", default: "#121314", second: "#6b6f7b", disabled: "#d6d8db" },
    "bg": {disabled: "#e4e5e7", brand: "#215bf0", brandHover: "#1146cc", brandSecond: "#ea1414", brandSecondHover: "#d40505" },
    "borders": {second: "#6b6f7b", brand: "#215bf0", dark: "#121314", disable: "#d6d8db", light: "#e4e5e7" },
    "function": {info: "#215bf0", error: "#f14646", warning: "#f0b621", success: "#4aa253" },
}
```

## Как этим пользоваться

После синхронизации токенов структура данных становится доступна в объекте темы.

Общий процесс такой:

Нужно задать цвет фона для некоторого элемента. В макете у этого элемента указан стиль:

![Стиль Figma](images/colors/style.png)

Вы знаете имя стиля и можете обратиться к соответствующей переменной темы:

```js
import { useTheme } from '@greensight/gds';

const SomeComponent = () => {
    const theme = useTheme();
    return (
        <button type="button" css={{ backgroundColor: theme.colors.brand }}>
            Button
        </button>
    );
};
```
