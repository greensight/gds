# Container

`Container` - это обёртка для ограничения контента по ширине. Этот компонент стилизован через значения ширины контейнера и полей, полученных через токены и хранящихся в объекте темы.

Контейнер используется в компоненте `Section`, но может использоваться и напрямую в других местах, где тег section не нужен.
